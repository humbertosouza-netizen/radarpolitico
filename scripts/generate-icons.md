# Gerar Ícones do PWA

Para gerar os ícones necessários para o PWA, você pode:

1. **Usar um gerador online**: https://realfavicongenerator.net/
2. **Criar manualmente**: Use um editor de imagens para criar:
   - `icon-192.png` (192x192px)
   - `icon-512.png` (512x512px)

**Design do ícone:**
- Fundo: #050B16 (grafite azul profundo)
- Logo: Olho com radar circular em neon verde (#4CFF85)
- Estilo: Minimalista, reconhecível em tamanhos pequenos

**Coloque os arquivos em:**
- `public/icon-192.png`
- `public/icon-512.png`
